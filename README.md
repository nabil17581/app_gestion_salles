Application Gestion des Salles 
=================================

Cette application de gestion de salles est développée en Python avec une interface graphique (GUI) et utilise MySQL pour la gestion des données. Elle repose sur une architecture en couches, permettant de séparer clairement l’accès aux données, la logique métier et l’interface utilisateur. Cette structure rend l’application plus organisée, maintenable et évolutive.
